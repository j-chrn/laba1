📖 Справка по программе

📋 Описание программы

Программа предназначена для учета и управления личными доходами. Позволяет добавлять, удалять, просматривать и сохранять информацию о доходах.

🔘 Главное меню

При запуске программы отображается главное меню с тремя кнопками:

  • 💼 РАБОТАТЬ - Переход к окну управления доходами с таблицей.
  • 📖 СПРАВКА - Открывает это окно с подробной информацией о программе.
  • 🚪 ВЫХОД - Закрывает программу (с подтверждением).

💼 Рабочее окно (Таблица доходов)

После нажатия кнопки "РАБОТАТЬ" открывается окно с таблицей доходов.

Функции рабочего окна:

  • ➕ Добавить - Добавляет новый доход в таблицу.
  • 🗑️ Удалить выделенный - Удаляет выбранный доход из таблицы.
  • 🔄 Обновить из файла - Перечитывает данные из файла incomes.txt.
  • 💾 Сохранить в файл - Сохраняет все текущие данные в файл incomes.txt.
  • 🔙 НАЗАД В МЕНЮ - Возвращает в главное меню.

📝 Формат данных

Каждый доход описывается следующими полями:

  • Дата - Дата получения дохода (ГГГГ.ММ.ДД)
  • Источник - Наименование источника дохода
  • Сумма - Размер дохода в рублях (целое число)
  • Цвет - Цвет для категоризации

Пример записи в файле incomes.txt:

Доход: 2024.01.15 "Зарплата" 50000 "Синий"

💡 Подсказки

  • Данные автоматически не сохраняются - не забывайте нажимать кнопку "Сохранить в файл" после изменений!
  • Для изменения размера окна перетащите границы
  • В поле "Дата" автоматически подставляется текущая дата

📁 Файл данных

Программа работает с файлом incomes.txt, который находится в той же папке, что и программа.

📌 Важно: Файл должен быть в кодировке UTF-8. Каждая запись находится на отдельной строке.
